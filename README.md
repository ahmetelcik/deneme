Scss Special Reset
===================

divler için ayarlayabileceğiniz class'lar

## Kullanımı: padding: { value }
```
<div class="pa-10">
  Lorem ipsum dolor sit amet, consectetur adipisicing elit
</div>
```

## padding-left: { value } Kullanımı
```
<div class="p-l-10">
  Lorem ipsum dolor sit amet, consectetur adipisicing elit
</div>
```

### Kullanımı 
```
<div class="p-l-0 p-t-10 p-b-10 p-r-10">
  lorem
</div>
```
